## Definition
Universal Email Alerts deliver real-time automated notifications that track performance changes, technical issues, and ranking movements across all connected projects in Search Atlas.

## How it works
Consolidates alerts from across the Search Atlas ecosystem (OTTO SEO, Site Auditor, Keyword Rank Tracker, GSC Performance) into one unified notification system. Each alert includes contextual data: affected URL, keyword, or campaign, plus direct access links to the corresponding dashboard module.

## Alert categories
- Ranking fluctuations
- Indexing issues
- Core Web Vitals changes
- Backlink activity
- Local performance updates
- AI automation status reports

## Core capabilities
- Customizable alert frequency, priority thresholds, and recipient lists
- Cross-tool synchronization (OTTO detects a metadata conflict, Site Auditor identifies an indexing drop)
- AI-prioritized impact scoring for high-severity events
- Fully white-labeled, client-facing alerts with branded dashboards

## Definition
Website Studio is an AI Landing Page Builder inside Search Atlas that generates complete, conversion-ready landing pages from a single prompt. Pages are created, structured, and published with SEO-ready architecture, conversion sections, tracking, form capture, and CDN delivery in place.

## How it works
User describes the desired page in plain language (product, audience, location, style, colors, key elements). Website Studio generates the complete page structure, then supports editing through commands or visual controls before publishing to CDN.

## Performance claims (from product page)
- 10x faster page production
- 90+ Lighthouse scores out of the box (performance, SEO, accessibility, best practices)
- 100% structured marketing pages generated instantly
- 0 developer dependency

## Core capabilities
- **Prompt-Based Page Creation**: complete marketing pages from a single input
- **Clone Website Mode**: recreate the structure and visual style of any page
- **PPC Landing Page Generator**: campaign-specific pages aligned with ad intent
- **Static CDN Hosting**: global edge network, free SSL auto-provisioned
- **AI Page Editing & Iteration**: commands or visual controls
- **Lead Capture Forms**: built into page structure
- **Tracking Script Support**: GA4, Google Tag Manager, Meta Pixel, custom scripts
- **Custom Domain & Instant Publishing**: CNAME connection, SSL provisioned automatically
- **Semantic HTML, optimized meta tags, schema markup**: applied automatically
- **White-Label Website Builder**: fully branded pages under agency name
- **Version restore**: return to any previous page version

## Industry use cases
- PPC campaign pages
- Agency client pages
- Local business pages
- SaaS product pages
- E-commerce campaign pages
- Online course pages

## Getting started (3 steps)
1. Describe the page in a single prompt
2. Edit with commands or visual controls (optional)
3. Publish and launch instantly on CDN-hosted pages

## Availability
Included in all Search Atlas plans (Starter through Agency).

## Related concept
Vibe coding: the practice of directing AI to build or edit pages through natural language commands. Website Studio applies this to landing page creation.

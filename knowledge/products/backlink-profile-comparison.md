## Definition
Backlink Profile Comparison Tool is a competitive SEO analytics system that compares backlink metrics across 2 to 6 domains and quantifies link equity distribution.

## How it works
Provides side-by-side measurements pulled from the Search Atlas backlink index (over 100 trillion links and 500 million domains).

## Core capabilities
- Side-by-side comparison of Domain Authority (DA), Page Authority (PA), linking domains, referring pages, anchor text ratios
- Link type classification: follow, nofollow, sponsored
- Dofollow and nofollow link ratios
- Domain diversity analysis
- Referring-page volume
- Interactive summary graphs for link quality, link velocity, authority growth

## Export and reporting
- One-click export integration with Report Builder

## Definition
Meta Generator is an AI metadata generation tool that constructs optimized meta titles, descriptions, and tags through semantic weighting, keyword intent modeling, and entity correlation.

## How it works
Uses contextual NLP mapping to align metadata with page content, search intent, and ranking signals from live SERP datasets. Applies keyword prioritization (primary weighting, secondary modifiers, contextual synonyms) and entity anchoring (brand, product, location).

## Length thresholds
- Titles: 50 to 60 characters
- Descriptions: 150 to 160 characters

## Core capabilities
- Single-page or bulk metadata generation
- Alignment with Google Search and Open Graph formatting standards
- Entity-linked metadata aligning with schema attributes and Knowledge Graph structures
- Bulk processing with multilingual context preservation

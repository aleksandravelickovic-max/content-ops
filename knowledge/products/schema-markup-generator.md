## Definition
Schema Markup Generator creates structured data in JSON-LD format (JavaScript Object Notation for Linked Data) for pages including blogs, products, and business listings.

## How it works
User selects schema type, enters page-specific details, and the tool generates properly formatted markup that search engines can parse for rich results eligibility.

## Supported schema templates
- Organization
- LocalBusiness
- Product
- Review
- FAQ
- HowTo
- Article

## Core capabilities
- Custom property mapping and field-level modification (name, URL, price, rating, publication date)
- Automatic schema version updates aligned with Google documentation and validation rules
- AI-assisted field completion
- Batch schema creation
- Google validation sync
- Automatic JSON-LD injection via OTTO pixel for real-time deployment

## Integrations
- OTTO SEO (pixel injection)
- Content Genius

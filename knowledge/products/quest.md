## Definition
QUEST is an AI-powered LLM visibility engine in Search Atlas that identifies which documents LLMs reference, reveals influence pathways, and uncovers visibility gaps.

## How it works
Analyzes any topic, brand, or keyword query. Generates full LLM responses with the complete AI-generated answer plus extracted source URLs, domain contributors, and editorial origins.

## Core capabilities
- Cross-model analysis (GPT, Claude, Perplexity, and others)
- Extracted LLM citations per response
- Source document lists
- Contributor identity data
- Visibility relevance scoring
- Automatic clustering of recurring source documents into priority clusters aligned with AEO (Answer Engine Optimization)

## Use cases
- Identify LLM-visible opportunities
- Correct outdated AI responses
- Influence documents shaping LLM output

## Integrations
- Publisher Exchange
- Digital PR Tool
- Link Building Outreach (QUEST PR Finder)

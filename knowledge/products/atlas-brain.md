## Definition
Atlas Brain is the conversational AI SEO agent inside Search Atlas. It reports and executes full SEO strategies from a single conversational interface, rather than stopping at recommendations.

## How it works
The user types a request in natural language. Atlas Brain interprets the goal, decomposes it into concrete actions (audits, optimizations, campaign steps, fixes), and executes them directly across connected Search Atlas modules with approval checkpoints where risk matters.

## Core capabilities
- **Conversational execution layer**: tasks run from chat, not buried in dashboards or menus.
- **Playbooks**: pre-structured agent workflows for specific use cases (for example, Smart Ads campaign creation, audit-and-optimize flows, LLM visibility audits).
- **Two operating modes**:
  - **Fast mode**: speed and minimal setup for users who want quick outcomes.
  - **Advanced mode**: step-by-step approvals and tighter governance at each stage.
- **Natural language to actions**: goals translate into concrete, executable steps across modules.
- **Human-in-the-loop gating**: approval checkpoints on high-impact changes.
- **Cross-module execution**: runs inside Smart Ads, OTTO SEO, LLM Visibility, Website Studio, and other Search Atlas products.

## Positioning
Other AI tools today are recommendation engines that generate advice or text and leave execution to the user. Atlas Brain performs the tasks directly from the conversation.

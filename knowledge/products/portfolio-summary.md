## Definition
Portfolio Summary is an AI-powered multi-account performance engine that generates unified scoring across connected accounts.

## How it works
Analyzes all connected accounts and produces a weighted 0 to 100 health score per account. Incorporates visibility changes, traffic signals, ranking trends, and engagement metrics.

## Per-account output
- Health score (0 to 100)
- Week-over-week change
- Category assignment: Biggest Wins, Stable, At Risk
- Linked performance summary

## Core capabilities
- Filtering by brand, region, performance category, or integration source
- Automatic smart categorization based on momentum patterns
- Cross-account scoring (not isolated project dashboards)
- LLM visibility included alongside organic metrics
- Daily refresh cadence

## Integrations
- Report Builder
- Keyword and Content workflows
- GA4, GSC, GBP
- LLM Visibility

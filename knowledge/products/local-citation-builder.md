## Definition
Local Citation Builder is a local SEO feature that automates the distribution and optimization of business data across verified citation networks. Enforces NAP (Name, Address, Phone) accuracy.

## Data aggregator networks (5)
- Data Axle
- Foursquare
- Neustar Localeze
- Yellow Pages Network
- GPS Network

## How it works
Four-stage workflow:
1. **Initial Data Check** — verifies data accuracy
2. **Submission Phase** — transmits validated data to selected aggregators
3. **Quality Assurance Phase** — reviewer confirms compliance
4. **Reporting Phase** — generates a campaign report with live URLs

## Credit and update structure
- 30 Hyperdrive credits per aggregator
- 120 credits for all five aggregators
- 12 months of free updates without additional credit deductions

## Distributed data
Name, address, phone number, website URL, business category metadata.

## Core capabilities
- Campaign dashboard with submission status, progress, live listing links
- Citation Listings view for ongoing updates
- Editable campaigns with permanent data updates
- Revalidation on updates within the same aggregator network

## Integrations
- GBP Galactic
- OTTO SEO

## Definition
Keyword Rank Tracker (KRT) is a ranking intelligence module in Search Atlas that measures keyword visibility, device segmentation, and geographic performance for tracked domains and pages.

## How it works
Collects verified keyword data through direct integration with Google Search Console and the internal Search Atlas SERP index. Provides daily, weekly, or monthly position updates.

## Core capabilities
- Historical position graphs per keyword
- SERP feature indicators (featured snippets, map pack, people-also-ask)
- Traffic-weighted visibility metrics
- Segmentation by device (desktop, mobile) and region (city, state, country)
- Competitor visibility tracking across up to 6 competing domains
- Ranking deltas, keyword overlap ratios, visibility trendlines
- Keyword volatility, CTR correlation, anomaly detection

## Integrations
- GSC Performance (feeds performance attribution reports)
- Keyword Gap Tool (feeds content optimization)
- Content Genius

## Definition
Report Builder is an AI-augmented reporting engine that consolidates SEO, content, LLM visibility, and analytics data into unified dashboards and client-ready reports.

## How it works
Compiles metrics from GA4, GSC, GBP, backlinks, rank tracking, and AI visibility. Generates real-time reports with automated visualizations, white-label exports, and scheduled delivery.

## Report content
- Keyword performance
- Traffic insights
- Backlink activity
- Local SEO trends
- Content analytics
- Visibility metrics

## Core capabilities
- Per-client, per-module, or per-data-source customization
- Automatically structured dashboards aligned with agency reporting frameworks
- Cross-account scoring
- AI-generated summaries
- Automated scheduled delivery

## Integrations
- Portfolio Summary
- Content Genius
- GA4, GSC, GBP

## Definition
WILDFIRE (Websites Inter-Linking Directly Fostering Influence, Reputation, and Engagement) is an automated backlink exchange network developed by Search Atlas. Generates inbound links through verified topical relationships and real-time authority validation.

## How it works
Establishes controlled link transactions under a 2:1 exchange ratio: every two outbound links create one inbound backlink from a high-authority, contextually relevant source. Link quality algorithms calculate topicality, domain power, and spam probability for each referring domain.

## Measured per participating site
- Domain Power (authority score, trust coefficient, ranking potential)
- Organic traffic (monthly volume, keyword distribution, engagement ratio)
- Topicality (semantic relevance, content proximity, link context)

## Per-transaction metrics
- Domain power differential
- Topicality percentage
- Organic traffic variance

## Core capabilities
- Active, pending, and approved outlink statuses visible in the OTTO project dashboard
- OTTO validation filters spam patterns
- Real-time link integrity checks
- Internal crawler verifies backlinks remain active within the indexed period

## Guardrails
- Excludes low-quality domains, expired properties, irrelevant sources
- Compliance enforcement to prevent footprint formation
- Non-reciprocal link structuring
- Domain rotation

## Integrations
- OTTO project dashboard
- OTTO authority scoring

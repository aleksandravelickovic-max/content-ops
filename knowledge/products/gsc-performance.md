## Definition
GSC Performance Tool is an integrated analytics environment inside Search Atlas that visualizes keyword rankings, traffic fluctuations, and page performance through synchronized Google Search Console (GSC) and Google Analytics 4 (GA4) data.

## How it works
Aggregates verified API data through an OAuth connection, which validates domain ownership and imports canonical GSC metrics (total clicks, impressions, average position). Pairs those values with GA4 traffic analytics to calculate real-time visibility changes and organic value estimates.

## Core capabilities
- Search behavior metrics: queries, impressions, CTR
- Engagement signals: sessions, bounce rate, on-page value
- 31-day rolling metrics for ranking trajectory, impression stability, CTR distribution
- Pages module: indexable URLs, depth, internal link count
- Keywords module: traffic share, CPC, MSV (monthly search volume)
- Cannibalization detection: identifies overlapping URLs competing for the same query
- Site Events overlay: correlates algorithm updates, technical deployments, or structural edits with ranking changes
- Historical trend charts for keyword positions, page traffic, index coverage

Website Studio supports direct injection of tracking scripts into any page:

- GA4 (Google Analytics 4)
- Google Tag Manager (GTM)
- Meta Pixel
- Custom scripts

No code changes are required on the underlying page. Full analytics and retargeting pipelines run natively.
